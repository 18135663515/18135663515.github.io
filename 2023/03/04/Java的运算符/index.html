Java的运算符

发表于2023-03-04|更新于2023-03-04
|阅读量:

算术运算符

（+ 、 - 、 * 、 / 、 % 、 ++ 、 --）

注意事项

1、整数相除只能得到整数,想要得到小数,必须有小数参与运算
2、字符串参与加法运算,实际上做的是拼接效果
3、字符串参与加法运算,实际是拿肌肤在计算机中存储的数据值来参与运算的(可以参照阿斯克码表)

阐述%(取余)和除法的区别

1、取余(%):去两个数相除的余数【如果是小数取余大数 取不尽 则得到那个小数】
2、除法(\):两个数相除时候得到的商

阐述++ 和 –的作用 以及 ++ –在前或者在后的区别

++:自增(+1)           --:自减(-1)
单独使用++ --,在前或者在后,结果都是一样的
1、++在前(进行其他运算):先进行自增,在进行运算或赋值操作
2、++在后(进行其他运输):先进性赋值或者运算,再进行自增
【--与++一样的规则】

关系运算符

（== 、 != 、 > 、 <  、 <= 、 <=）
关系运算符的结果是布尔类型的(boolean)只有true 和 false
PS:不要把关系运算分的==和赋值运算符的=搞混

逻辑运算符

逻辑运算符是用于连接关系表达式的
&(与)、|(或)、^(异或)、|(非)、&&(双与)、||(双或)
  • &:有false则false
  • |:有true则true
  • ^:相同为false 不同为true
  • !:结果为true则false 泽国为false则true
  • &&:具有短路效果,前面为false则后面不进行判断
  • ||:具有短路效果,前面为true则后面不进行判断

三元运算符

格式：数据类型 变量名 = 关系表达式?表达式1 : 表达式2;
  • 执行流程
    1、先计算表达式的结果 是true还是false
    2、如果是true,就把表达式1返回当结果————如果是false,就把表达式2返回当结果
    int a = 10;
    int b = 20;
    ing c = (a > b)? a : b; (false , c = 20)
    int d = (a < b)? a : b; (true , c = 10)

赋值运算符

基本的赋值运算符：=
常用的赋值运算符：+= 、 -= 、 *= 、 /= 、 %=

int a = 10;
a += 20;(a = a + 20)

  • PS:常用的赋值运算符隐含了强制转换特性
