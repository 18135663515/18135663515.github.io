MySQL的基础查询

发表于2023-03-03|更新于2023-03-04
|阅读量:

基础查询语句

SELECT * FROM 表名; //查询表中所有的列
SELECT 列名1 , 列名2... FROM 表名; //查询指定列信息
SELECT DISTINCT 列名 FROM 表名; //去除重复记录查询(关键字：DISTINCT)
SELECT 列名 AS 别名 FROM 表名; //别名查询(关键字：AS)
SELECT * FROM 表名 AS 别名; //别名查询(关键字：AS)
SELECT 列名 , 数据类型的列名 + 数字 FROM 表名; //运算查询

【数据类型的列名就是 可以进行数学数字运算的类型 , 字符串类型的进行运行算结果为NULL】

条件查询语句

BETWEEN...AND...(显示在某一区间的信息,含头含尾)
IN(1 , 2 , 3...)(包含条件)
LIKE 通配符(%匹配多个字符 LIKE'a%a')(_匹配单个字符'a_')
     例：王大力、张老三、红警蓝天、红警08、魔鬼  （LIKE'%天'  可以找到 红警蓝天）（LIKE'魔_'  可以找到魔鬼）
IS NULL(判断为空)
IS NOT NULL(判断不为空)
AND(多条件成立)   
OR(多条件 任一成立)
NOT(不成立)
