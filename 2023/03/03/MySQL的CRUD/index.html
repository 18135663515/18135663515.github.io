MySQL的CRUD

发表于2023-03-03|更新于2023-03-04
|阅读量:

创建库 创建表

CREATE DATABASE 库名; //创建库
CREATE TABLE 表名 (字段名 , 类型 , 约束); //创建表
DROP TABLE 表名; //删除表
SHOW TABLES; //查案看所有数据表
SHOW DATABASES; //查看所有数据库
DESC 表名; //查看表结构
USE 库名; //切换数据库

修改表结构

ALTER TABLE 表名 ADD 列名 类型 约束; //添加列
ALTER TABLE 表名 MODIFY 列名 类型 约束; //修改列
ALTER TABLE 表名 CHANGE 旧列名 新列名 类型 约束; //修改列名
ALTER TABLE 表名 DROP 列名; //删除列
ALTER TABLE 表名 CHARACTER SET 字符集; //修改表的字符集 
RENAME TABLE 表名 TO 新表名; //修改表名

插入表数据、更新表记录、删除表记录

INSERT INTO 表名 (列名1 , 列名2 , 列名3...) VALUES (值1 , 值2 , 值3...); //插入某些列
INSERT INTO 表名 VALUES (值1 , 值2 , 值3...); //插入所有列 【必须有主键约束】

UPDATE 表名 SET 字段名 = 值 , … WHERE 条件; //修改某一条,不带WHERE条件则修改全部

DELETE FROM 表名 WHERE 条件; //删除表记录

修改条件的写法

=(等于)  <>(不等于)  <=(小于等于)    
AND(Java里的双与'&&')       OR(Java里的双或'||')
NOT(Java里的非'!')     IN(1 , 2 , 3...)(包含条件,包含其中一个)

注意：DOS命令窗口下操作数据库会报错,表数据中文乱码【set names’gbk’】 临时改变DOS的编码值
