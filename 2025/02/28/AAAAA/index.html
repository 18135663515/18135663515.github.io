5. 常见问题

❓ 如何选择最佳方案？

  • 日常文档 → 小白版
  • 批量转换 → 极客代码版
  • 合同/票据 → OCR增强版+DeepSeek API

❓ OCR识别精度如何提升？

  1. 扫描分辨率 ≥ 300dpi
  2. 优先使用DeepSeek API（准确率98%+）
  3. 中文文档添加 lang='chi_sim' 参数

❓ 转换后排版错乱怎么办？

  • 使用Adobe Acrobat进行后期排版调整
  • 复杂表格推荐使用 DeepSeek表格增强版
